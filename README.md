# Ames
Decentralized and free messenger

## Built with
* Nuxt.js
* Vue.js
* Express.js
* Babel
* NeDB

## Installation
Here is a simple Ames setup:
```bash
git clone https://github.com/Asocio/Ames.git
npm install
npm start
```
