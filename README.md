Decentralized and free messenger

## Built with
* Nuxt.js
* Vue.js
* Express.js
* Babel
* NeDB

## Start Ames
Here is a simple Ames start:
```bash
git clone https://bitbucket.org/amesproject/ames.git
npm install
npm run build
npm run start
```

Node verison: 8.x+
