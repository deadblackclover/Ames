## Built with
* Nuxt.js
* Vue.js
* Express.js
* Babel
* NeDB

## Documentation
[ames-documentation](https://asocio.github.io/ames-documentation/)

## Start Ames
Here is a simple Ames start:
```bash
git clone https://github.com/Asocio/Ames.git
npm install
npm start
```
Node verison: 8.x
